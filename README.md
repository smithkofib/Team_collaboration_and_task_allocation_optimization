1. **Clone the repository**:

   ```bash
   git clone https://github.com/smithkofib/Team_collaboration_and_task_allocation_optimization.git
   cd team_task_allocation_proj
  

2. **Create a virtual environment (optional but recommended):**
   python -m venv team_collaboration_and_task_allocation_env
   
   source team_collaboration_and_task_allocation_env/bin/activate 

3. **Install dependencies:**
   pip install -r requirements.txt

4. **Run migrations:**
   python manage.py migrate

5. **Run the development server:**
   python manage.py runserver
